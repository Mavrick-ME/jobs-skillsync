## 🎯 Overview

Jobs SkillSync addresses critical inefficiencies in traditional recruitment by integrating
Natural Language Processing and Machine Learning. Instead of keyword matching, it uses
**BERT-based semantic embeddings** to understand the true meaning of resumes and job descriptions.

---

## ✨ Features

- 🧠 **Semantic Matching** — BERT understands context, not just keywords
- 📄 **Smart Parsing** — Supports PDF, DOCX and TXT resumes
- ⚖️ **Bias Mitigation** — Anonymizes names, gender, age and location
- ⚡ **Blazing Fast** — 0.233s per resume, 1000+ resumes/minute
- 🏆 **Ranked Results** — Candidates scored and ranked by match percentage
- 🔐 **Role-based Auth** — Separate recruiter and admin portals
- 🗄️ **Database** — SQLite stores all jobs and match history
- 🎨 **Modern UI** — Dark futuristic design with Spline 3D and particle animations
